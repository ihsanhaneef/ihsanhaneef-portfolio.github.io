CH ifthar'24

Ifthar'24 is an website for the students to register online for suhoor and ifthar so that the organizers can arrange the food beforehand. The student's who are having fast during the Ramadan need to have the food, which will provided by the student's organization. And the students need to register online through the website so that the committee get the accurate number of students who need food. This website is created for the Iftar registration. I've built this website using HTML, CSS, Bootstrap and JavaScript, featuring a responsive design for mobile and desktop viewing. I've developed the backend using Node.js implementing online registrations, updates and other details. The admin can view, show and update the details of users.
